Creating fixed-width import filters

To create an import filter for fixed-width data input files:

  1. Locate the import filters xml-file. This file can usually be found in C:\..\HEC-DSSVue\Plugins\ImportFilters.xml.
  2. Open the xml-file in a text file editor or a dedicated xml file editor.

  3. Create a new <format> node entry below the last <format> node following the structure in the table below:

    Node Sub-node Description Example value
    name An arbitrary filter name. River Board format (".txt")
    fileName extension The file name extension of the input data files. txt
    firstDataLine The line number in a data file containing the first record of data. 6
    fileFormat type Specify 0 for fixed-width data file import filters. 0
    date yearStartChar First character position within a line in the data file for a record's "year". 1
    date monthStartChar First character position within a line in the data file for a record's "month". 5
    date dayStartChar First character position within a line in the data file for a record's "day". 7
    time hourStartChar First character position within a line in the data file for a record's "hour". 9
    time minuteStartChar First character position within a line in the data file for a record's "minute". 11
    value startChar First character position within a line in the data file for a record's data value. 23
    value endChar Final character position within a line in the data file for a record's data value. 37
    qualityFlag startChar First character position within a line in the data file for a record's quality flag (optional). 38
    qualityFlag endChar Final character position within a line in the data file for a record's quality flag (optional). 42
    qualityFlag missingValueFlag Flag indicating missing values (optional). M
    qualityFlag missingValueValue Parameter value to be reset to "missing value" (optional). -999
