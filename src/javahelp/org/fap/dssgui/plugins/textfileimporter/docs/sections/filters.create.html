Creating import filters

Import filters are defined in an Extensible Markup Language (XML) file. This file can be found in: C:\..\HEC-DSSVue\Plugins\ImportFilters.xml. XML files are well-structured, plain text files which can be edited by text editors or dedicated xml-editors.

Two different types of input filters can be defined:

  1. For fixed-width input data files (i.e. each column has a set width).
  2. For delimited input data files (i.e. each column is separated by for example a comma).

Depending on the import filter type, different configuraton parameters need to be specified within the import filters xml-file.
