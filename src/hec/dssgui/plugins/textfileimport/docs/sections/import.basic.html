Basic time series importing

To import one or more time series datasets into HEC-DSSVue:

  1. Move of copy all input data files into a single folder using Windows Explorer. Read more on preparing input data.
  2. In the HEC-DSSVue main window, click on the "Import..." button on the toolbar. A time series import window opens.
  3. Select the import filter in the drop down box corresponding to the format of the input data. Read more on creating new import filters.
  4. Select the folder in which the input data files are located.
  5. Click the "Import" button.
  6. Analyse the diagnostic messages in the message box.
  7. If the importing was succesfully close the import window.
  8. Verify the imported data in the main HEC-DSSVue window.
